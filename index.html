About

D365 Configuration Navigator — Designed by Zamir Gori

Project overview
D365 Configuration Navigator, a visual, narrative-driven tool envisioned and developed by Zamir Gori, is designed to simplify and humanize the complex process of configuring a new Microsoft Dynamics 365 Finance & Operations legal entity. Inspired by editorial storytelling, it transforms dense configuration checklists into an intuitive, magazine-like experience.
Content source
All 44 configuration tasks are derived from the Microsoft Dynamics 365 F&O New Legal Entity Configuration Checklist (Supply Chain Organization, Version 1.0, April 2025). Sample answers reference a representative company: Contoso Supply Co.
Design system
Clinical SaaS Blueprint — cool whites and electric blue with a tight grid and data-dense layout. Geist typeface. Optimized for consultant workflows: fast search, expandable detail cards, KPI summaries, and breadcrumb navigation.
Usage notes
Mandatory items are required for go-live. Optional items are best practices. All sample answers must be validated with client stakeholders. Costing methods are legally binding once transactions exist.
Implementation sequence notes
01
Follow the configuration sequence strictly — Org Admin → Finance → SCM → Inventory. Foundation must be complete before subsequent modules.
02
All "Sample Answers" are illustrative defaults. Review and confirm with client business stakeholders during Fit-Gap Analysis workshops.
03
Number sequences should be reviewed against local regulatory requirements before finalizing (consecutive invoice numbering for tax authority compliance).
04
Costing methods (Standard, FIFO, Weighted Average) are legally binding once inventory transactions exist. Confirm with Finance before configuration.
05
Advanced Warehouse Management (WMS) licensing must be confirmed with Microsoft before enabling location/license plate tracking.
06
Security roles and data access should be configured in parallel. Data migration (COA, Item master, Vendor/Customer master) tracked separately.
